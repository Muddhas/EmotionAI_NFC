===============================================================
NTAG 213 芯片注入指南 — 使用 TagWriter App (手机端)
===============================================================

前置准备:
  1. Android 手机安装 NXP TagWriter App
     - Google Play 搜索 "NXP TagWriter" 下载
     - 或从 NXP 官网下载 APK
  2. 确保手机 NFC 已开启
  3. 准备空白 NTAG 213/215/216 芯片
  4. 将 ntag213_tagwriter.csv 文件传到手机

===============================================================
方法一: 导入 CSV 批量写入 (推荐)
===============================================================

1. 打开 TagWriter App
2. 点击右上角 ⋮ 菜单 → "Import from..."
3. 选择 CSV 文件所在位置 (下载目录 / 文件管理器)
4. 选择 ntag213_tagwriter.csv
5. App 解析后显示记录列表
6. 点击第一条记录 → "Write" → 将芯片贴近手机背面
7. 写入成功会有提示
8. 点击下一条记录 → 放下一张芯片 → 写入
9. 重复直到所有芯片写入完成

===============================================================
方法二: 手动逐条写入
===============================================================

1. 打开 TagWriter App
2. 点击 "Write" → "New" → "URI"
3. 输入: emoai://chip/EMO-213-00000001
4. 点击 "Next" → 将芯片贴近手机背面
5. 写入成功后有提示音
6. 换下一张芯片和 chipId，重复步骤 2-5

建议写入后记录 chipId 和芯片的对应关系:
  ┌──────┬──────────────────────────┐
  │ 芯片 │ chipId                   │
  ├──────┼──────────────────────────┤
  │ #1   │ EMO-213-00000001          │
  │ #2   │ EMO-213-00000002          │
  │ #3   │ EMO-213-00000003          │
  │ ...  │ ...                       │
  └──────┴──────────────────────────┘

===============================================================
验证写入
===============================================================

1. TagWriter → "Read"
2. 贴近已写入的芯片
3. 确认读取到的 URI 为: emoai://chip/EMO-213-XXXXXXXX

===============================================================
⚠️ 注意事项
===============================================================

- 每张芯片写入后建议用标签贴标记 chipId，便于后续使用
- NTAG 213 无防伪功能，安全性仅靠服务端账号绑定机制
- 如果 TagWriter 提示 "Tag not empty"，先执行擦除 (Erase)
- 如果 CSV 导入格式错误，尝试备选格式:
  - 格式2: uri,,emoai://chip/EMO-213-00000001 (空 subtype = 无前缀)
  - 格式3: text,en,emoai://chip/EMO-213-00000001 (纯文本记录)
- CSV 中的 `none` 表示 URI 前缀代码 0x00 (不添加任何前缀)
- 如果 TagWriter 提示格式错误，手动写入时选择 URI 类型，填写完整的 emoai://chip/xxx
- 213 芯片演示的是「账号绑定」流程，而非防伪
